Keys Tables

Editing Mode: Lights

Key

Function

F1

Cycle primary Camera view / Switch out of Edit mode

F4

Enter Edit mode

SCROLL LOCK

Toggles the speed at which you move/drag objects, using the mouse or movement keys.
SCROLL LOCK light OFF=normal speed / SCROLL LOCK light ON=high speed

LEFT
MOUSE-BUTTON

1. Cursor movement / Selection. Float the cursor over the object, then use ENTER to select it.
2. Object 'dragging'. Click and hold on a pre-selected object, then use the movement keys or Right Mouse-Button to drag the object through the world (works better in Camera View).

RIGHT
MOUSE-BUTTON

1. Click and hold for "freelook" mode.
2. Edit Mode: Lights only: Change foucs beam of selected Spot type lights. Select the Spot Light source using ENTER. Click and hold the Right Mouse-Button and then use the mouse movement to move the (focus) line. Press ENTER again to deselect the source (when cursor is not over source) - re-enabling 'freelook' on the Right Mouse Button (1.)

Q

Move backwards (-Z)

W

Move Up (+Y)

E

Move Forward (+Z)

A

Move Left (-X)

S

Move Down (-Y)

D

Move Right (+X)

INSERT

Insert new Light

SPACEBAR

Toggle Light Type: Omni / Omni Normal / Spot / Spot Normal / Square Shadow

O
(o for orange)

Turn Selected Light On/Off (useful to judge the effect a light's having)

F

Toggle Flicker value: On/Off

-

Decrease Flicker Speed value

+

Increase Flicker Speed value

ENTER (RETURN)

Select / Deselect Light

(Left) SHIFT

1) Clone Light: Select Light to clone (ENTER), hold (left)SHIFT and left-mouse drag away from the Light to create a duplicate.
2) Speeds up incriment/decrease when used with incrimental settings keys (Eg: SHIFT & NUMPAD 9)

TAB

Cycle through View Axis: XY, XZ, ZY, X, Y, Z

(Left) ALT

Toggles between Camera/World view option (movement Axis)

DELETE

Delete currently selected Light

NUMPAD /
(Spot Types Only)

Decrease Cone (width) value

NUMPAD *
(Spot Types Only)

Increase Cone (width) value

NUMPAD -

Decrease Reach value

NUMPAD +

Increase Reach value

NUMPAD ENTER

Toggle environment effect for selected Light: World Only / Objects Only / World & Objects

NUMPAD 4

Decrease Red value of Light

NUMPAD 5

Decrease Green value of Light

NUMPAD 6

Decrease Blue value of Light

NUMPAD 7

Increase Red value of Light

NUMPAD 8

Increase Green value of Light

NUMPAD 9

Increase Blue value of Light

RIGHT
MOUSEBUTTON
(Spot Types Only)

Once Spot Light is selected and placed, use the Right Mouse button to move the focus of the beam along your selected X,Y or Z axis.

(Left) CTRL & F4

Save changes to Lighting file (.LIT)
